================================================================================
STEP-BY-STEP VERCEL SETUP
================================================================================

1. After importing your GitHub repo to Vercel, you'll see:
   "Configuring Project"

2. Click "Continue" (default settings are fine)

3. Wait for it to ask for Environment Variables
   (Or go to Settings ‚Üí Environment Variables)

4. Click "Add Environment Variable"

5. For each variable above:
   a) Paste the NAME from this file
   b) Paste the VALUE from this file
   c) Select environments: Production, Preview, Development
   d) If PRIVATE variable: Check "Sensitive" box
   e) Click "Save"

6. After adding all 5 variables, click "Deploy"

7. Wait 2-5 minutes for deployment to complete

AFTER DEPLOYMENT
================================================================================

1. Copy your Vercel URL (e.g., https://grief-companion-v2.vercel.app)

2. Go to Supabase dashboard:
   https://supabase.com/dashboard/project/rfqbxfduvvfwhjocxeqf

3. Navigate to: Authentication ‚Üí URL Configuration

4. Update the "Redirect URLs" field to include:
   https://YOUR_VERCEL_DOMAIN/auth/callback

   Example:
   https://grief-companion-v2.vercel.app/auth/callback

5. Click "Save"

Then you're ready to test!
